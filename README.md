# Project-Manager-Dashboard-Web-Application
The Project Manager Dashboard is a web-based application designed to help users efficiently manage their projects. It allows new users to register and log in securely, and provides full control over project data through an intuitive dashboard interface.

Features-
User Registration & Secure Authentication
Login / Logout Functionality
Create New Projects
Update Existing Project Details
View Project Information
Delete Projects
User-Friendly Dashboard UI
CRUD Operations for Project Management

Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
